Getting started
===============

Install ``snecs`` the same way as any `PyPI
<https://pypi.org/project/snecs>`_ package. Depending on the dependency
manager you're using for your project:

`pip <https://pip.pypa.io/en/stable/>`_:

.. code-block:: console

   $ pip install snecs

`Pipenv <https://pipenv.pypa.io/en/latest/>`_:

.. code-block:: console

   $ pipenv install snecs

`Poetry <https://python-poetry.org/>`_:

.. code-block:: console

   $ poetry add snecs
